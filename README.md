# Real-Time API Data Integration & Automation (Live Data)
### Built a real-time, automated Power BI dashboard that visualizes global development indicators using live data from the World Development Indicators API. 
Developed end-to-end Python pipelines to continuously fetch, clean, and transform API data into structured datasets, which were integrated into Power BI for analysis. 
This solution combines Python,  APIs, and Power BI to deliver continuous monitoring of key global development indicators such as GDP, trade, health expenditure, internet penetration, unemployment, and poverty trends.

## Project Workflow
     1. API Extraction with Python
        - Uses requests library to pull data from the World Development Indicators API.
     2. Data Cleaning & Transformation
        - Cleaning missing values and inconsistent formats and merging multiple indicator datasets.
     3. Live Data Integration with Power BI
        - Power BI connected to Python script.
     4. Interactive Dashboard (Power BI)
        - DAX measures for averages, growth rates, comparisons, and correlations.
